# Ruby Devenv

The Vagrant box, fully prepared to run as a Ruby development environment.

## What is inside?

`Ubuntu 14.04`

* postgres-client
* js-runtime(nodejs)
* docker 1.11.0
* docker-compose
* ruby 2.3
* bundler
